Commands

  • !forgetmenot <emoji/emote> Sets the reaction emoji that triggers saving quotes
  • !forgetmenot <local/global> Sets quotes to be local to channels, or globally available in a server
  • !quote [person/*] [search term] Get a random quote based on optional criteria
  • !remember Remembers the replied-to message, or the most recent message in the channel

Permissions

  • Anyone with "Manage Messages" permission can add quotes
  • Anyone with "Adminitrator" can run !forgetmenot commands
  • Anyone can run the quote view command
